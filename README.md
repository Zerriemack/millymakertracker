# millymakertracker

## What is this?

Milly Maker Tracker is a web project built for NFL daily fantasy sports research. It helps users study DraftKings Milly Maker trends, review winning lineup patterns, and use a simulator to explore slate outcomes in a live web environment. For this first milestone, the main shipped feature is the simulator page, which is deployed and publicly viewable.

## Why does it exist?

This project exists to give DFS players a cleaner way to study tournament information and make better lineup decisions. Instead of digging through scattered contest data, users can view lineup trends, compare slate information, and interact with a simulator built around the way large field NFL tournaments are played. The goal is to turn research into something easier to access, easier to understand, and more useful during lineup preparation.

## What tools did I use?

I built this project with Astro because it gave me a strong structure for a content-heavy site with custom pages and fast navigation. I used TypeScript and JavaScript for the simulator logic and front-end interactivity. I used CSS to shape the dark visual style and keep the site consistent across pages. The project is stored on GitHub for version control and deployed on Vercel so the site can be viewed online. I also used AI support during development to help debug problems, refine layout decisions, and move faster through implementation while still keeping the project aligned with my own goals and design direction.
